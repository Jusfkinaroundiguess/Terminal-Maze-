#Terminal maze game to play if you're bored while coding!
#If you wish to play the game then simply clone the given 
repository and run the main.py file.
To change or  modify the maze, head to the maze.py file 
and edit the variable sample_maze. 

Follow the given 
guidelines while editing or creating a new maze:
-Start with a line of underscores which defines
the roof of the maze (floors are the same)
-For "paths" where the player can go: use spaces, such as 
leaving a blankline after the initial "roof" and then a "floor".
This is necessary as the algorithm uses this information to 
figure out where the player can go and not go.
-Finally, use pipes for vertical walls in the maze.
